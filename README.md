# CareerPilot AI

CareerPilot AI is an intelligent career mentor that helps students analyze their skills, identify gaps, and generate personalized career roadmaps.

## ✨ Features
- Skill Gap Analysis
- Career Readiness Score
- Job Level Detection
- Personalized Career Advice
- Roadmap Generator
- Project Suggestions
- Interview Questions

## 🛠 Tech Stack
- Java
- Spring Boot
- HTML, CSS, JavaScript
- Bootstrap
